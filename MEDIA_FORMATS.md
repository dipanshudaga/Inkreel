# Inkreel Media Formats

Inkreel categorizes content into specific formats based on the source data (TMDB, Google Books, AniList). This ensures that your collection accurately reflects the medium of each entry.

## 1. Cinema & TV (via TMDB)
| Format | Description |
| :--- | :--- |
| **Feature Film** | standard movies released in theaters or on streaming. |
| **TV Series** | Episodic content with multiple seasons. |
| **Miniseries** | Limited series designed for a single season run. |

---

## 2. Anime & Manga (via AniList)
AniList provides more granular format tracking for Japanese media:

- **TV / TV Short**: Standard broadcast anime series.
- **Movie**: Animated feature films (e.g., Studio Ghibli works).
- **Special**: One-off episodes or short side-stories.
- **OVA**: Original Video Animation (direct-to-video releases).
- **ONA**: Original Net Animation (web-first releases).
- **Music**: Animated music videos.
- **Manga / One Shot**: Serialized or standalone comic works.
- **Novel**: Light novels or prose works.

---

## 3. Literature (via Google Books)
| Format | Description |
| :--- | :--- |
| **Paperback / Hardcover** | Standard physical book formats. |
| **E-Book** | Digital publications. |
| **Audiobook** | Narrative audio versions. |

---

## 4. How Formats are Displayed
In the **Watch** and **Read** sections, formats are displayed as secondary metadata (e.g., "Feature Film", "TV Series"). For Anime, the app uses AniList's specific terminology to preserve the cultural context of the release.
